Cómo usar la salida de productos

1) Consulta automática

Podés usar la lupa dentro del código (o F2) para buscar como en Ingreso/Ajuste. Elegí la bodega e ingresá el código: al terminar de escribirlo (o al cambiar bodega o marca) se cargan los datos del producto y el stock. Consultar producto repite la consulta manualmente.

2) Marca y cantidad

Elegí la marca desde la lista o escribila; el sistema sugiere marcas ya usadas en bodega. Indicá la cantidad a retirar según el máximo mostrado debajo del campo. Si hay variantes, definí marca y bodega o usá Usar esta fila en la tabla inferior.

3) Registrar

Completá la observación si querés dejar motivo, cliente o referencia interna. Pulsá Guardar salida para descontar el stock; el sistema valida que haya existencia suficiente.
