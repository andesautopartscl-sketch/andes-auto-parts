Ayuda del ajuste de stock

Consulta

  • Tocá la lupa dentro del campo de código (o F2) para buscar como en Ingreso: código, descripción u OEM.
  • Elegí la bodega e ingresá el código: al terminar de escribirlo (o al cambiar de bodega) se consulta el saldo automáticamente.
  • Podés usar Consultar stock si preferís repetir la consulta.
{% if mostrar_ajuste_multi %}

Varias variantes

  • En la tabla ajustá el nuevo stock por marca; + Agregar variante suma filas para otras marcas.
  • Pulsá Aplicar ajuste una sola vez: se guardan todos los cambios de la tabla en un solo movimiento.
{% else %}

Aplicar

  • Revisá el nuevo stock en bodega y, si aplica, la observación (motivo del ajuste).
  • Pulsá Aplicar ajuste para registrar el movimiento y el diferencial.
{% endif %}

Incorporación a producto

  • Elegí motivo Incorporación a producto y el código destino (ej. AR240RC).
  • El stock del origen debe bajar. Si el destino ya tiene ingresos con costo, se prorratea el valor y se recalcula el P. venta manteniendo el margen.
  • Si el destino aún no tiene costo, el movimiento queda registrado y el monto a sumar se indica para el próximo ingreso.
